What is Swift poetry?

It's poetry written in a Swift language.

The programming languages are just like human languages. They can be used to express emotions, thoughts, ideas, personalities. They can be used to communicate with another human beings.

They can be used to write poetry.

But they also compile and execute.

So the Swift poetry is the poetry on two levels: that the code means, and how the code executes.
